# C++ Core：公開裁罰資料檢索與排序模組

<!-- 說明這個 C++ 模組的定位 -->
本模組是 JobCheck 專案中的 C++ 資料結構與演算法核心。Python Flask 系統負責網頁查詢與視覺化，C++ Core 則使用同一份公開資料，獨立實作資料檢索、統計與排序功能，用來展示 C++、資料結構、演算法與時間複雜度分析能力。

---

## 模組定位

<!-- 說明為什麼不是把 Python 全部改成 C++ -->
C++ Core 不是取代 Python Flask 網站，而是作為同一專題中的核心能力補強：

- Python：資料清理、網頁介面、視覺化、使用者操作
- C++：資料索引、快速查詢、統計排序、資料結構實作
